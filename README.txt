Rock Paper Scissors Game

This is a simple Python game. You type rock, paper, or scissors and the
computer picks one too. The game tells you if you win, lose, or tie.

How to run the game without Docker:
1. Open the folder with the files.
2. Run the main.py file in PyCharm

How to run the game using Docker:
1. Open a terminal.
2. Go to the folder with the files.
3. Build the image with: docker build -t rpsgame .
4. Run the game with: docker run -it rpsgame
